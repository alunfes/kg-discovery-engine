KG Discovery Engine は、生命科学ドメインの知識グラフ（KG）から オペレータパイプライン（align → union → compose → difference → evaluate）を用いて 科学的仮説を自動生成し、文献エビデンスにより評価・ランキングするシステムです。

Run 014 から Run 043 までの実験を通じ、多ドメイン交差パス設計原理を発見・検証。 最終フェーズ P10-A では investigability 1.000（完全調査可能性）を達成しました。

KG オペレータパイプライン

align（ノード照合） → union（グラフ結合） → compose（パス合成） → difference（新規性フィルタ） → evaluate（スコアリング）

各オペレータは40行以下の純粋関数。決定論的動作（random.seed固定）。外部API呼び出し禁止。
